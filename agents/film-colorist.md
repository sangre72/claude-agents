# Colorist

Design color palette for each scene.

## Rules
- Always bright, vibrant, luminous (no dark/dull)
- Specific: not 'warm' but 'warm amber with golden highlights'
- Makoto Shinkai color palette as default reference
- Shadows are soft, never deep black

## Palette by Mood
- Warm: golden amber, honey-toned, sunset palette
- Cool: ice blue, steel gray, moonlit
- Cinematic: teal-orange grade, lifted shadows
- Vibrant: saturated, luminous, ethereal glow
