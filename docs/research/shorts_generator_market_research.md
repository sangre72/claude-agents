## 4. 차별화 포인트 분석

### 4.1 로컬 설치형의 장단점

| 구분 | 장점 | 단점 |
|------|------|------|
| **데이터 보안** | 영상이 외부 서버로 전송되지 않음. 기업 보안 정책 충족 | - |
| **처리 속도** | 로컬 GPU 활용 시 빠른 처리. 인터넷 속도 무관 | GPU 없는 환경에서는 매우 느림 |
| **오프라인** | 인터넷 없이도 기본 기능 사용 가능 | AI 기능은 결국 API 필요 |
| **커스터마이징** | 고객 환경에 맞춘 깊은 커스터마이징 가능 | 설치/업데이트/호환성 관리 부담 |
| **비용 구조** | 서버 운영비 없음 (개발사 측) | 고객 PC 성능에 의존 |
| **지원 부담** | - | OS별 호환성, 설치 오류 등 기술 지원 비용 높음 |
| **확장성** | - | 팀 협업, 클라우드 공유 기능 구현 복잡 |

**핵심 판단**: 로컬 설치형은 **기업/대행사 B2B** 시나리오에서 데이터 보안 측면의 강점이 있으나, 일반 크리에이터에게는 설치 허들이 큰 단점. 브루(Vrew)가 이 모델로 한국 시장에서 성공한 선례가 있음.

### 4.2 BYOK(Bring Your Own Key) - AI 비용 고객 직접 부담 모델

**장점:**
- **가격 투명성**: 고객이 AI 사용량을 직접 확인하고 제어 가능
- **마진 구조 개선**: 개발사는 AI API 비용을 부담하지 않아 수익성 예측 가능
- **기업 고객 선호**: 대기업은 이미 AI 제공업체와 계약이 있어 기존 계약 활용 가능
- **모델 선택 자유**: 고객이 원하는 AI 모델(GPT-4, Claude, 로컬 LLM 등) 선택 가능
- **비용 확장성**: 사용량 증가에 따른 비용 부담이 고객에게 분산

**단점:**
- **진입 장벽**: API 키 발급/관리가 비기술 사용자에게 어려움
- **사용자 경험**: 별도 결제 구조가 복잡하게 느껴짐
- **총비용 불확실**: "도구비 30만원 + AI 비용 ?"로 총비용 예측 어려움
- **경쟁사 대비**: SaaS 경쟁사는 올인원 가격으로 단순함. BYOK는 복잡해 보임
- **지원 문제**: AI API 오류 시 도구 문제인지 API 문제인지 구분 어려움

**핵심 판단**: BYOK 모델은 **기술 친화적인 대행사/기업** 타겟에는 합리적이지만, 일반 사용자 대상으로는 마찰이 큼. JetBrains, Warp 등 개발 도구에서 BYOK가 성공하는 이유는 타겟이 개발자이기 때문.

### 4.3 가능한 차별화 방향

1. **데이터 보안 + 기업 맞춤**: "영상이 외부로 나가지 않는 유일한 숏폼 도구" 포지셔닝
2. **워크플로 자동화**: 기획서 -> 대본 -> 영상 -> 숏폼 -> 멀티플랫폼 배포 일괄 파이프라인
3. **한국어 특화**: 한국어 자막/자연어 처리 정확도에서 해외 도구 대비 우위
4. **대량 처리**: 대행사용 배치 처리, API, 화이트라벨 지원
5. **AI 모델 유연성**: BYOK로 최신 모델 즉시 적용 가능 (경쟁사는 모델 교체에 시간 소요)

1. **시장 성장**: AI 영상 편집 도구 시장 CAGR 42%, 숏폼 시장 지속 성장
2. **한국 시장 공백**: 해외 도구는 한국어 지원 약함, 국내 도구(브루)는 숏폼 자동 생성 특화 아님
3. **B2B 수요**: 대행사/기업의 영상 마케팅 비용 증가 추세
4. **로컬 설치 차별화**: 데이터 보안 민감 기업 대상 유일한 포지션
5. **BYOK 트렌드**: JetBrains, Warp 등 개발 도구에서 검증된 모델

### 6.2 위험 요인

1. **가격 저항**: 월 30만원은 경쟁사 대비 3~17배 비쌈. 강력한 차별화 없이는 판매 어려움
2. **빠른 기능 평준화**: AI 기능은 빠르게 복제됨. 오늘의 차별화가 내일은 사라질 수 있음
3. **무료/저가 대안**: Opus Clip 무료 플랜, 브루 무료 플랜 등 강력한 무료 대안 존재
4. **설치형 한계**: 팀 협업, 자동 업데이트, 멀티디바이스 지원에서 SaaS 대비 불리
5. **BYOK 마찰**: 비기술 사용자(자영업자, 일반 크리에이터)에게 API 키 관리는 높은 진입 장벽
6. **타겟 시장 협소**: 30만원 지불 가능 + 기술 친화적 + 숏폼 대량 제작 수요 = 교집합이 매우 좁음

### 6.4 기획팀장에게 전달 사항

**필수 포함 기능:**
- 긴 영상 -> 숏폼 자동 클리핑 (핵심 가치)
- AI 자막/캡션 자동 생성 (한국어 고정확도)
- 다중 플랫폼 포맷 지원 (9:16, 1:1, 16:9)
- 브랜드 템플릿 관리 (B2B 필수)
- 배치 처리 (대행사 핵심 니즈)

**선택 기능:**
- 바이럴 점수 예측
- AI B-roll 자동 삽입
- 소셜미디어 자동 게시 연동
- API 제공 (대행사/개발자용)
- 화이트라벨 (대행사 리셀링)

**제외 권장:**
- 영상 촬영/생성 기능 (범위 초과)
- 일반 영상 편집 기능 (브루와 직접 경쟁 회피)

**핵심 권고:**
1. **가격 재검토 필수**: 월 30만원 단일 가격은 시장에서 수용되기 어려움. 3단계 가격 체계 + BYOK 분리 권장
2. **타겟 명확화**: "마케팅 대행사와 기업 마케팅팀"으로 한정. 개인 크리에이터/자영업자는 현 가격에서 제외
3. **ROI 메시지**: "편집자 1명 인건비(300만원+)로 무제한 숏폼 생성"이라는 가치 제안 필요
4. **한국어 특화**: 해외 도구의 한국어 자막 정확도가 낮은 점을 공략
5. **PoC 우선**: 대행사 2~3곳과 파일럿 후 가격/기능 검증 후 본격 출시 권장
